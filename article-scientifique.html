L'avènement de l'industrie du Big Data

Extrait de Open Data & Big Data : Nouveaux défis pour la vie privée, de Danièle Bourcier et Primavera de Filipi, 2016

Un écosystème Big Data est apparu du fait des avancées technologiques rapides, au cours des dernières décennies, en matière de stockage de données, d’informatique et de modélisation scientifique.

L’objectif de l’industrie du Big Data est de mettre au jour des relations, inattendues mais utiles, pouvant prédire des résultats s’avérant bénéfiques aux consommateurs, à l’entreprenariat et à la société. Les analystes du milieu estiment que le Big Data deviendra un marché évalué à 50 milliards de dollars d’ici 2017.

La collecte des données

Les consommateurs, les entreprises ou encore les gouvernements produisent constamment des données. Chaque passage de carte de crédit, chaque bouton « like » de Facebook, chaque courrier électronique envoyé, ou chaque page Web visionnée produisent des données. Les dispositifs connectés à Internet permettent la saisie et le stockage d’une quantité considérable d’émissions numériques suit aux activités effectués sur Internet.

Lors des cinq années écoulées entre 2006 et 2011, la quantité de données a augmenté d’un facteur de neuf. Presque toutes les données sont à vendre.

Pam Dixon et Robert Gellman du World Privacy Forum ont identifié 186 variables connues que les courtiers en données sur la consommation vendent aux agrégateurs et analystes. Trois types de données, autrefois indisponibles aux prêteurs, présentent désormais des entrées potentiels pour les modèles d’évaluation de crédit :

  • Web browsing data : Les données de navigation Web - sur internet.
  • Geolocation data : Les données de géolocalisation.
  • Social network data : Les données de réseaux sociaux.

Les données de navigation sur le Web

En tapant une adresse URL ou en cliquant sur un lien, un internaute demande une page, transmettant son adresse IP (Internet Protocol) au serveur hôte pour la page Web. Une adresse IP constitue un identifiant unique, semblable à une empreinte numérique. Le serveur répond à la requête et téléchargé une ligne de texte appelée un cookie sur le disque dur de l’internaute, ou crée une "empreinte numérique" si ce dernier utilise un téléphone portable.

Diverses techniques peuvent exploiter les fréquences radios les téléchargements d’applications mobiles ou encore les déviations dans l’accéléromètre d’un téléphone portable. Ces différentes techniques en question révèlent des données telles que le type d’appareils utilisé, les préférences de shopping ainsi que les recherches Google effectuées. Les prêteurs ont identifié, par exemple, le type de téléphone portable comment enta un indicateur potentiel de solvabilité.

Alors que les cookies et les empreintes numériques tracent le cheminement Web d’un internaute, les téléphones portables connectés à internet transmettent des données qui peuvent tracer le chemin physique, concert d’un utilisateur. Les entreprises peuvent tracer le lieu de résidence de leurs clients, où ils jouent et où ils travaillent. Les compagnies de téléphones portables et les institutions spécialistes en données fournissent des données de géolocalisation d’une manière cumulée, mais il est possible de suivre des individus à travers les applications mobiles. Les entreprises n’ont même pas besoin d’obtenir le consentent d’un utilisateur avant la vente des données en question.

Outre les données sur les positions physiques et virtuelles, les sites de réseaux sociaux assurent de nos jours une vue enrichie sur les relations d’une personne. Le comportement d’un internaute sur les sites de réseaux sociaux peut révéler des réponses à certaines questions que se poserait un agent de crédit. Pour illustrer, des chercheurs de l’Université de Cambridge ont découvert qu’en analysant les boutons « j’aime » ils peuvent, de manière fiable, prédire la race, la sexualité, la religion l’affiliation politique et certains traits de personnalité. À condition qu’un utilisateur Facebook « aime » la page « je soutiens mon président », mais « n’aime pas » la page officielle de « David Bowie », les probabilités que cet internaute soit africain-américain sont fortes par exemple.
